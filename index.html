Welcome to the Data Processing Project!

This project demonstrates a robust data processing pipeline using Python, Pandas, and GitHub Actions. We process raw data from data.xlsx (converted to data.csv) and generate summarized insights in result.json.

The entire process, from linting with Ruff to generating and publishing the final results, is automated via a GitHub Actions CI/CD workflow. The result.json is automatically published to GitHub Pages, making the processed data easily accessible.

Key Features:

  • Automated data processing with Python & Pandas.
  • Code quality enforced with Ruff linter.
  • Continuous Integration/Deployment with GitHub Actions.
  • Automated publication of results to GitHub Pages.

Input Data

Our source data originates from data.xlsx, which is converted into a more CI-friendly data.csv format for processing.

Processing Script

The core logic resides in execute.py, a Python script designed to perform aggregations and transformations using the Pandas library.

Generated Results

The final processed data is outputted as result.json. This file is automatically published via GitHub Pages.
